# 基金技术指标

## 夏普比率（Sharpe Ratio）

### 定义与核心思想
夏普比率由诺贝尔经济学奖得主威廉·夏普于1966年提出，是衡量投资"性价比"最经典的标准化指标之一。它衡量的是：**投资者每承担一单位总风险，能获得多少超额回报（超越无风险利率的部分）**。

打个比方：两条船要从上海开到洛杉矶，A船开得快但一路颠簸得让人晕船，B船慢一些但航线稳健几乎不晃。夏普比率衡量的就是"每晃一度，能跑出多少净速度"——即**单位风险带来的超额收益**。

### 计算公式
$$
夏普比率 = \frac{E(R_p) − R_f}{σ_p}
$$

其中：
- $E(R_p)$：投资组合的预期收益率（实践中常用历史平均收益率替代）
- $R_f$: 无风险利率（通常用国债利率或定期存款利率）
- $σ_p$: 投资组合的标准差，即总风险（波动率）

计算示例：若无风险利率为3%，基金A预期收益率为19%，标准差为8%，则`夏普比率 = (19% − 3%) / 8% = 2`，意味着每承担1%的风险，可获得2%的超额收益。

实践中通常采用"事后夏普比率"，即用历史平均收益率替代理论公式中的预期收益率。评估时应尽量拉长时间周期（如三年、五年），以降低短期偶然性的干扰。

### 参考价值
| 夏普比率范围 |  含义与参考价值                    |
| :--------- | :------------------------------ |
| > 1        | 收益水平显著高于风险，性价比良好      |
| 0 ~ 1      | 收益高于无风险利率，但性价比一般      |
| < 0        | 收益跑输无风险利率，不如买国债或存银行 | 

夏普比率大于1通常被视为不错的表现，它帮助投资者从"只看收益率"转变为综合考量风险与收益，避免追逐高风险、高收益但性价比低的产品。此外，高夏普比率往往意味着波动较低、走势稳健，投资体验更好，也更容易享受时间的复利效应。

### 局限性
- 必须在同类型基金中比较，不能拿股票型基金和债券型基金对比
- 基于历史数据计算，不代表未来表现
- 使用标准差衡量风险，假设收益服从正态分布（实际未必满足）
- 结果受时间跨度和无风险利率选择的主观影响


## 信息比率（Information Ratio）

### 定义与核心思想
信息比率是衡量**主动管理能力**的核心指标，由Richard Grinold和Ronald Kahn在20世纪80年代末的"主动投资管理基本法则"中进行了数学定义和推广。

与夏普比率不同，信息比率关注的是**相对于特定业绩基准**的超额收益。它衡量的不是"绝对值"，而是"相对值"——**基金经理每承担一单位主动风险（即偏离基准的风险），能创造出多少超越基准的超额回报**。

通俗理解：它回答了"我主动偏离基准去搏收益，每一分偏离的风险换回了多少超额回报"。

### 计算公式
$$
信息比率 = \frac{R_p − R_b}{TE}
$$

其中：
- $R_p$: 投资组合收益率
- $R_b$: 业绩基准收益率（如沪深300指数）
- $TE$: 跟踪误差，即($R_p$ - $R_b$)序列的标准差（也称为主动管理风险）

计算示例：某基金年度回报率为15%，基准指数为10%，基金收益率与基准收益率差异的标准差为10%，则`信息比率 = (15% − 10%) / 10% = 0.5`，即每承受1个单位的超额风险，可获得0.5个单位的超额回报。

计算时需选择合适的基准，确保基准能真实反映投资组合的投资范围、风格和风险特征（如美股大盘基金采用S&P 500），并保持数据频率一致（如月度、年度），最终进行年化处理。

### 参考价值
| 信息比率范围 |  含义与参考价值                   |
| :--------- | :------------------------------ |
| > 0        | 基金经理跑赢业绩基准               |
| < 0        | 基金经理跑输业绩基准               | 
| 正值且越高   | 以相对较小的主动风险换取较大超额收益 | 

信息比率为正值代表基金经理采用的策略能获得高于基准的回报，正值越大，说明每一单位主动管理风险创造的超额收益越高。通常，信息比率在**0.2～0.5被视为尚可，0.5以上视为优秀**。

信息比率的应用场景包括：管理人筛选、业绩归因（评估创造Alpha的效率）、风险预算配置，以及跨周期、跨策略的超额表现对比。

### 局限性
- 依赖基准选择的合理性，不同投资组合基准不同，横向比较存在难度
- 信息比率相同的两个组合可能风险特征迥异，一个跟踪误差1%、一个15%，不能简单等同评价
- 同样基于历史数据，无法预测未来
- 实践中可采用"先分档再计算"的方法——按跟踪误差将基金划分为低、中、高风险档位，再在各档位内按信息比率排序，从而更公正地评价主动管理能力


## 实操中的使用建议
- 夏普比率帮你判断"这只基金整体是否值得投资"，信息比率帮你判断"基金经理是否真的创造了超越市场的能力"。一把尺子量到底，容易失之偏颇。
