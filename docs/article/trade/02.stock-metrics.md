# 股票技术指标

## 市值 (Market Cap)
**公式：股价 × 总股本**

如果你要把这家公司（比如贵州茅台）整个买下来，需要付多少钱。
- 大盘股（如银行、两桶油）：像大象，稳，但跑得慢。
- 小盘股（如微盘股）：像猴子，跳得高，但容易摔死。

## PE
**PE（市盈率）= 股价 ÷ 每股收益**，也可以理解为 **公司总市值 ÷ 年度净利润**。

它的核心含义是：按目前的盈利水平，你投入的钱需要多少年能回本。

- **高PE（如30倍、50倍以上）**：通常意味着市场对这家公司未来高增长有预期（如科技股、成长股）。但也可能是股价被过度炒作的泡沫。
- **低PE（如5倍、8倍以下）**：可能意味着公司被市场低估（如长期稳定的银行、公用事业股）。但也可能是公司未来前景暗淡、盈利即将下滑，市场给的“便宜是有道理的”。


## PB
**PB（市净率）= 股价 ÷ 每股净资产**，也可以理解为 **公司总市值 ÷ 股东权益**。

它的核心含义是：按目前的净资产水平，你投入的钱需要多少年能回本。

- PB = 1，说明你花 1块钱买了公司 1块钱的家当。
- PB < 1（破净），说明公司打折卖，通常见于银行、钢铁。

## ROE
**ROE（净资产收益率）= 净利润 ÷ 净资产**

它反映的是：公司用股东投入的每一块钱，一年能赚回多少钱。数值越高，说明公司赚钱效率越高，是巴菲特非常看重的指标。

- **高ROE（如15%-20%以上）**：通常意味着公司有某种护城河（品牌、技术、特许经营权等），是利润的“优等生”。比如一些白酒、家电龙头常年ROE在20%以上。
- **低ROE（如长期低于5%-8%）**：可能处于竞争激烈的行业，或公司经营不善。不过，有些周期型或高速扩张期的公司，ROE可能会暂时性偏低。

> ⚠️ **注意**：ROE需要结合负债率来看。一家公司如果借了大量债务（高负债率），也能把ROE做高，但风险也更高。


### ROE 结合 PE 选股策略
ROE和PE单独使用都有缺陷，最好结合起来看，形成互补：

- **筛选优质候选股：先用ROE**：把ROE长期（比如5-10年）稳定在 15%甚至20%以上 的公司找出来。这能帮你快速排除大量平庸或亏损的公司，聚焦于盈利能力强的企业。
- **确定买入时机：再用PE**：在ROE达标的公司池子里，对比历史PE、行业平均PE。选择在PE处于历史相对低位或合理区间时买入。这就避免了在ROE很高但PE也高得离谱的时候追高接盘。


## 成交量
成交量的核心价值在于，它是**衡量价格变动真实性和持续性的“燃料表”**。价格是“表象”，而成交量是“本质”。

### 成交量的两大核心参考价值
1. **验证趋势的“真伪”**：价格上涨时成交量放大，说明上涨有资金推动，趋势更可靠；反之，放量下跌则趋势更危险。
2. **识别市场的“拐点”**：价格创新高但成交量萎缩，说明上涨缺乏动力，可能见顶；价格创新低但成交量放大，说明下跌动能增强，可能见底。

### 什么是“量价背离”？
简单说，**量价背离就是价格与成交量的变化方向不一致**。它就像一个预警信号，提示当前趋势的推动力可能正在减弱。

通常分为两种情况：
- **顶背离 (看跌信号)**：价格创新高，但成交量没有同步放大，甚至萎缩。这常发生在上涨趋势末期，意味着追涨力量不足，是股价可能即将见顶回落的警示。
- **底背离 (看涨信号)**：价格创新低，但成交量没有继续放大，反而萎缩或趋于稳定。这常发生在下跌趋势末期，意味着恐慌性抛盘减少，是股价可能即将见底反弹的信号。

### 九种核心量价关系速查

| 量价组合 | 低位 (建仓区) | 中位 (拉升区) | 高位 (出货区) |
|---------|-------------|-------------|-------------|
| 量增价涨 | 主力进场，积极关注 | 趋势健康，坚定持股 | 边拉边出，警惕见顶 |
| 量增价跌 | 主力借利空洗盘，企稳后可低吸 | 多空分歧，破位则离场 | 主力出货，果断离场 |
| 量缩价涨 | 筹码锁定好，主力控盘，持股待涨 | ⚠️ 量价背离，上涨动力不足，警惕回调 | 诱多出货，逢高离场 |
| 量缩价跌 | 抛压耗尽，地量见地价，等待放量 | 洗盘尾声，缩量回踩关键支撑可低吸 | 阴跌不止，坚决远离 |
| 量平价涨 | 买盘稳定，趋势可能延续 | 缺乏爆发力，但趋势尚可 | 多方力量减弱，需警惕 |
| 量平价跌 | 持续卖出，但抛压不重 | 趋势偏弱 | 阴跌，市场无承接 |

## 换手率
**换手率**（Turnover Rate），也称**周转率**，是指在特定时间内，**某只股票的累计成交股数与其流通股本的比例**。

简单理解就是：市场上“换手”或“转手”买卖该股票的比例。它衡量的是股票交易活跃度的核心指标。

- **高换手率**：意味着股票交易非常活跃，买卖双方分歧大，流动性好。
- **低换手率**：意味着股票交易冷清，关注度低，流动性差。

### 计算公式
计算公式非常简单：

$$
换手率 = \frac{某段时间内的成交量（股）}{该股票的流通股本（股）} \times {100\%}
$$

- **成交量**：该时间段内成交的总股数。
- **流通股本**：上市公司在市场上可以自由交易的股份总数（扣除限售股）。

**举例**：
某股票流通股本为 1 亿股，今天总共成交了 2000 万股，则当日换手率为：`2000万 / 1亿 × 100% = 20%`。

### 参考价值
区分不同阶段的行情性质

| 股价阶段      |  换手率特征                   | 市场含义                                        |
| :----------  | :-------------------------  | :--------------------------------------------- |
| 低位横盘      |  换手率极低（<1%）             | 无人关注，筹码可能正在被主力悄悄收集。               |
| 温和上涨      |  换手率温和放大（3%-7%）       |  资金开始介入，量价配合健康，上涨有持续性。            |
| 加速拉升      |  换手率持续维持高位（>10%）     |  多空分歧加大，但人气爆棚，短线机会多，风险也逐步累积。  |
| 高位巨量      |  单日换手率突然暴增（>15%-20%） |  很可能是主力在对倒出货，风险极高，需高度警惕。        |
| 下跌初期      |  换手率仍然较高                |  恐慌盘和抄底盘共同作用，但下跌趋势通常难以逆转。      |
| 长期下跌后    |  换手率重新萎缩至极低水平        |  杀跌动能衰竭，可能正在构筑中期底部。                |

## 龙虎榜

### 定义 
龙虎榜是沪深证券交易所每个交易日收市后，公布的当日出现交易异常波动的证券名单。榜单的核心内容是：异动股票的交易价格、成交量，以及买入和卖出金额最大的前五家证券营业部（或机构专用席位） 的具体数据。

这一制度的建立初衷是提升市场透明度，为中小投资者打破资金交易的信息壁垒，帮助其了解大资金的实际动向；同时，公开信息披露本身也是一种威慑，便于监管部门快速锁定异动交易背后的资金主体，提升对违法违规交易的查处效率。

### 上榜条件
满足以下任一条件的股票即可上榜，但交易所通常只选取排名前列的股票：

| 上榜条件              |  沪深主板                | 创业板/科创板 |
| :------------------- | :--------------------- | :----------  |
| 涨跌幅偏离值           | 单日≥±7%                |  单日≥±15%   |
| 换手率                | ≥20%                   |  ≥30%        |
| 价格振幅              | ≥15%                    | ≥30%        |
| 连续3日涨跌幅偏离值累计 | ≥±20%                   | ≥±30%        |
| ST/ * ST股票         | 偏离值≥±5%（连续累计±12%） | 要求更严      |

各板块的差异体现了对不同定位的考量：科创板和创业板因本身波动性较高，上榜门槛相应提高；科创板每个条件选取前5名，披露面更广，与其支持硬科技企业的定位相匹配。

### 龙虎榜对投资者的价值（积极面）

#### 1. 揭示主力资金的真实动向
龙虎榜的核心价值在于揭示主力资金的动向和博弈格局。通过榜单，投资者可以了解到哪些资金在买入、哪些在卖出，直观地观察到机构、游资、外资之间的博弈。

#### 2. 判断资金性质，辅助决策逻辑
- **机构集中买入且金额较大**：通常代表专业机构认可该股的中长期价值。机构往往提前做好详细的研究和操盘计划，其行为具有趋势性和一定的前瞻性，机构净买入超过1亿元的个股尤其值得关注。
- **游资席位频繁现身**：往往意味着短线情绪爆发，但也伴随剧烈波动。不同游资的风格迥异，有的擅长"一日游"，有的则善于引导题材行情。
- **机构与游资同步看好**：即"合力榜"，筹码分布相对均衡，上涨动力的持续性可能更强；而单一席位买入过多，形成"一家独大"，后期往往面临抛压风险。

#### 3. 发现市场热点与板块主线
如果某一板块的多只股票频繁登上龙虎榜，说明该板块受到市场的集中关注，大概率是当前阶段的市场主线。投资者可以据此研判市场合力的方向。

#### 4. 结合股价位置判断主力意图
- **股价低位 + 多个机构持续大额买入**：可能是主力在低位吸筹，后续上涨的概率较大；
- **股价高位 + 游资大额买入**：可能是主力在借机出货，风险极高，需高度警惕。
